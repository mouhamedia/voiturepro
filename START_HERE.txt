DÉMARRAGE RAPIDE
================

1. Installer dépendances:
   composer install
   npm install

2. Configurer:
   cp .env.example .env
   php artisan key:generate

3. Database:
   php artisan migrate

4. Compiler assets:
   npm run build

5. Lancer:
   php artisan serve

Puis accédez à: http://localhost:8000

UTILISATION ADMIN
=================

2. Cliquez /dashboard
