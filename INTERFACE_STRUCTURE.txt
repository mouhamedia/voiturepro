INTERACTIONS
════════════

1. SIDEBAR TOGGLE (Mobile)
   ☰ Clic → Sidebar overlay
   Lien cliqué → Sidebar fermeture auto

2. NAVIGATION ACTIVE
   Page courante → Ligne orange à gauche
                → Fond teinté orange

3. BOUTON RETOUR
   ◄ Clic → history.back()

4. DÉCONNEXION
   🔴 Clic → Confirmation dialog
           → POST /logout
           → Redirect /login

5. FORMULAIRES
   ❌ Erreur → Alert rouge avec liste
   ✅ Succès → Toast notification auto-hide (5s)

6. TABLEAUX
   Hover → Ligne surlignée
   Icônes → Info-bulles au survol
